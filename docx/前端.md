## 🚀 关键功能实现思路

### 1. 粒子地球与聚拢特效

传统的 `SphereGeometry` 无法实现复杂的粒子动画。我们需要使用 `THREE.BufferGeometry` 并编写**自定义着色器（Shader Material）**。

- **逻辑：** 预加载一张地球地形权重图（或通过 GeoJSON 采样），在球面上生成数万个顶点。
    
- **聚拢效果：** 在 Vertex Shader 中定义一个 `uProgress` 变量。
    
    - 当缩放（Zoom）变化时，通过 GSAP 改变 `uProgress`。
        
    - 顶点位置 Pfinal​=Porigin​+(Ptarget​−Porigin​)×uProgress。
        

### 2. 局部特写 (Fly-to 功能)

当点击某个区域时，相机需要平滑移动到该区域的正上方。

- **坐标转换：** 经纬度 (ϕ,λ) 转换为 3D 空间坐标 (x,y,z)。
    
    x=R⋅cos(ϕ)⋅sin(λ)
    
    y=R⋅sin(ϕ)
    
    z=R⋅cos(ϕ)⋅cos(λ)
    
- **动画：** 使用 `gsap.to(camera.position, { ... })`，同时更新 `OrbitControls.target` 使视角中心锁定在该区域。
    

### 3. 连接线与热力图

- **弧线：** 使用 `QuadraticBezierCurve3`（二次贝塞尔曲线）连接两点，并配合 `Texture.offset` 实现飞线流动感。
    
- **热力图：** 建议使用一张透明的 Canvas 作为纹理贴在球体上方，或创建另一层密集的粒子群，根据数据调整粒子的大小和颜色。

3. **粒子聚拢特效 (Shader)**：通过自定义着色器实现粒子随缩放（Zoom）动态向中心聚拢或扩散的视觉效果。
    

### 💻 核心实现代码

JavaScript

```
import * as THREE from 'three';
import { OrbitControls } from 'three/examples/jsm/controls/OrbitControls';
import gsap from 'gsap';

export default class EarthEngine {
    constructor(container) {
        this.container = container;
        this.scene = new THREE.Scene();
        this.params = { radius: 100, particleCount: 50000 };
        this.init();
        this.createParticleEarth();
    }

    init() {
        // 1. 基础场景设置
        this.camera = new THREE.PerspectiveCamera(45, window.innerWidth / window.innerHeight, 1, 2000);
        this.camera.position.z = 400;
        this.renderer = new THREE.WebGLRenderer({ antialias: true, alpha: true });
        this.renderer.setPixelRatio(window.devicePixelRatio);
        this.renderer.setSize(window.innerWidth, window.innerHeight);
        this.container.appendChild(this.renderer.domElement);

        this.controls = new OrbitControls(this.camera, this.renderer.domElement);
        this.controls.enableDamping = true;

        window.addEventListener('resize', () => {
            this.camera.aspect = window.innerWidth / window.innerHeight;
            this.camera.updateProjectionMatrix();
            this.renderer.setSize(window.innerWidth, window.innerHeight);
        });
    }

    createParticleEarth() {
        const positions = new Float32Array(this.params.particleCount * 3);
        const randomScales = new Float32Array(this.params.particleCount);

        for (let i = 0; i < this.params.particleCount; i++) {
            // 使用球面坐标系随机分布
            const phi = Math.acos(Math.random() * 2 - 1);
            const theta = Math.random() * Math.PI * 2;

            positions[i * 3] = this.params.radius * Math.sin(phi) * Math.cos(theta);
            positions[i * 3 + 1] = this.params.radius * Math.sin(phi) * Math.sin(theta);
            positions[i * 3 + 2] = this.params.radius * Math.cos(phi);
            randomScales[i] = Math.random();
        }

        const geometry = new THREE.BufferGeometry();
        geometry.setAttribute('position', new THREE.BufferAttribute(positions, 3));
        geometry.setAttribute('aScale', new THREE.BufferAttribute(randomScales, 1));

        // 核心 Shader：控制粒子聚拢与无损缩放
        this.particleMaterial = new THREE.ShaderMaterial({
            uniforms: {
                uTime: { value: 0 },
                uZoom: { value: 1.0 }, // 1.0 为正常，数值越大粒子越向中心聚拢
                uColor: { value: new THREE.Color(0x3399ff) }
            },
            vertexShader: `
                attribute float aScale;
                uniform float uZoom;
                uniform float uTime;
                varying float vAlpha;
                void main() {
                    // 实现聚拢逻辑：位置随 uZoom 线性偏移
                    vec3 targetPos = position * (1.0 / uZoom);
                    vec4 mvPosition = modelViewMatrix * vec4(targetPos, 1.0);
                    
                    // 粒子大小随距离相机的远近自动调整（保持视觉一致性）
                    gl_PointSize = (2.0 + aScale * 2.0) * (300.0 / -mvPosition.z);
                    gl_Position = projectionMatrix * mvPosition;
                    vAlpha = 1.0 / uZoom; 
                }
            `,
            fragmentShader: `
                uniform vec3 uColor;
                varying float vAlpha;
                void main() {
                    // 绘制圆形粒子
                    float r = distance(gl_PointCoord, vec2(0.5));
                    if (r > 0.5) discard;
                    gl_FragColor = vec4(uColor, vAlpha * (0.6 - r));
                }
            `,
            transparent: true,
            blending: THREE.AdditiveBlending,
            depthTest: false
        });

        this.earth = new THREE.Points(geometry, this.particleMaterial);
        this.scene.add(this.earth);
    }

    update() {
        this.controls.update();
        this.renderer.render(this.scene, this.camera);
        requestAnimationFrame(() => this.update());
    }
}
```

## 🔵 模块二：地理边界与陆地粒子映射 (Geo-Mapping & Boundaries)

本模块的目标是将上一个模块中纯随机分布的球体粒子，精准地“贴”到地球的陆地区域，并绘制出清晰的国家/地区边界线。

本模块的核心难点在于**地理坐标（经纬度）与 3D 空间坐标（XYZ）的转换**，以及**高性能地生成成千上万条边界线**。

### 💻 核心实现思路

1. **数据源**：我们需要一个 World GeoJSON 文件（包含国家边界的多边形数据）。
    
2. **粒子采样优化**：不再盲目生成 5 万个粒子，而是遍历 GeoJSON 的多边形，在多边形内部进行有色采样，确保粒子全部落在陆地上。
    
3. **边界线生成**：将 GeoJSON 中的 `LineString` 坐标点转换为 `Vector3`，利用 `THREE.LineSegments` 或更高级的 `Line2` 模块生成连续的边界。
    

### 💻 核心代码框架 (新增地理图层模块)

JavaScript

```
import * as THREE from 'three';
import * as d3 from 'd3-geo'; // 用于辅助地理计算（可选）

export default class GeoLayer {
    constructor(scene, earthRadius) {
        this.scene = scene;
        this.radius = earthRadius;
        this.group = new THREE.Group();
        this.scene.add(this.group);
        
        // 预加载 World GeoJSON 数据
        this.loadGeoJSON();
    }

    async loadGeoJSON() {
        const response = await fetch('/path/to/world.json'); // 替换为你的 GeoJSON 路径
        const geoData = await response.json();
        this.drawBoundaries(geoData);
    }

    // 1. 经纬度转 3D 坐标的核心公式
    latLonToVector3(lat, lon, radius) {
        const phi = (90 - lat) * (Math.PI / 180);
        const theta = (lon + 180) * (Math.PI / 180);

        return new THREE.Vector3(
            -radius * Math.sin(phi) * Math.cos(theta),
            radius * Math.cos(phi),
            radius * Math.sin(phi) * Math.sin(theta)
        );
    }

    // 2. 绘制边界线
    drawBoundaries(geoData) {
        const lineMaterial = new THREE.LineBasicMaterial({ 
            color: 0x00ffff, 
            transparent: true, 
            opacity: 0.5 
        });

        geoData.features.forEach(feature => {
            if (feature.geometry.type === 'Polygon' || feature.geometry.type === 'MultiPolygon') {
                const coordinates = feature.geometry.type === 'Polygon' 
                    ? [feature.geometry.coordinates] 
                    : feature.geometry.coordinates;

                coordinates.forEach(polygon => {
                    polygon.forEach(ring => {
                        const points = [];
                        ring.forEach(coord => {
                            // 调用转换公式
                            points.push(this.latLonToVector3(coord[1], coord[0], this.radius + 0.1)); // 稍微悬空避免闪烁
                        });
                        
                        const geometry = new THREE.BufferGeometry().setFromPoints(points);
                        const line = new THREE.Line(geometry, lineMaterial);
                        this.group.add(line);
                    });
                });
            }
        });
    }

    // [待完善]：如何让 EarthEngine 中的粒子只落在陆地上？
    // 思路：在 createParticleEarth 中，生成随机点后，使用 d3.geoContains(geoData, [lon, lat]) 校验
}
```

## 🟡 模块三：连接线（飞线）与热力图 (Fly-lines & Heatmap Layer)

本模块的目标是让球体“活”起来，展示动态的数据流动和地理信息密度。

本模块的核心难点在于**如何在 3D 曲面上绘制优雅的弧线**，以及**如何高性能地渲染成千上万个具有数据属性（颜色、大小）的数据点**。

### 💻 核心实现思路

1. **飞线 (Fly-lines)**：
    
    - 给定两个经纬度（起点、终点）。
        
    - 计算两点在 3D 空间中的坐标，并计算出它们之间的**中点（但在更高半径上）**，作为贝塞尔曲线的控制点。
        
    - 使用 `THREE.QuadraticBezierCurve3` 生成弧线路径。
        
    - _高级特效_：为了实现“流动”感，不能直接画线。我们需要在 Vertex Shader 中传入时间 `uTime`，在 Fragment Shader 中根据采样点在曲线上的比例（Attribute `aProgress`），让只有一部分线段发光，从而形成流动的光斑。
        
2. **热力图粒子 (Heatmap Points)**：
    
    - 数据通常是 `[{ lat, lon, value }, ...]`。
        
    - 不能为每个数据点创建一个 `THREE.Mesh`，性能会崩溃。
        
    - **核心方案**：使用 **`THREE.InstancedMesh`** (实例化网格)。创建一个基础的粒子几何体（如圆片），然后一次性绘制数万个实例。每个实例的位置、颜色和缩放由一个 4x4 矩阵数组控制，直接在 GPU 上计算。
        

### 💻 核心代码框架 (新增数据图层模块)

JavaScript

```
import * as THREE from 'three';
import gsap from 'gsap';

export default class DataLayer {
    constructor(scene, earthRadius) {
        this.scene = scene;
        this.radius = earthRadius;
        this.group = new THREE.Group();
        this.scene.add(this.group);
        
        // 用于存储飞线和热力图的引用，方便后续更新或删除
        this.flyLines = [];
        this.heatPointsMesh = null;
    }

    // 辅助：坐标转换 (复用 M2 逻辑)
    _latLonToVector3(lat, lon, r) {
        const phi = (90 - lat) * (Math.PI / 180);
        const theta = (lon + 180) * (Math.PI / 180);
        return new THREE.Vector3(
            -r * Math.sin(phi) * Math.cos(theta),
            r * Math.cos(phi),
            r * Math.sin(phi) * Math.sin(theta)
        );
    }

    // ==========================================
    // 1. 高性能实例化热力图点
    // ==========================================
    createHeatmapPoints(dataArray) {
        const count = dataArray.length;
        // 基础几何体：一个圆片
        const geometry = new THREE.CircleGeometry(0.5, 12); 
        
        // 自定义材质：支持从 Shader 接收不同的颜色
        const material = new THREE.MeshBasicMaterial({
            transparent: true,
            blending: THREE.AdditiveBlending,
            depthWrite: false,
            side: THREE.DoubleSide,
            vertexColors: true // 关键：启用顶点颜色支持
        });

        this.heatPointsMesh = new THREE.InstancedMesh(geometry, material, count);

        const dummyMatrix = new THREE.Object3D();
        const colors = new Float32Array(count * 3);
        const colorTool = new THREE.Color();

        dataArray.forEach((item, i) => {
            // 1. 设置位置和方向（面朝球心）
            const pos = this._latLonToVector3(item.lat, item.lon, this.radius + 0.5);
            dummyMatrix.position.copy(pos);
            dummyMatrix.lookAt(new THREE.Vector3(0, 0, 0)); // 始终面朝地球
            
            // 2. 根据数据 Value 设置大小（无损缩放的一部分）
            const scale = 1.0 + item.value * 5.0; // 自定义映射逻辑
            dummyMatrix.scale.set(scale, scale, 1);
            
            dummyMatrix.updateMatrix();
            this.heatPointsMesh.setMatrixAt(i, dummyMatrix.matrix);

            // 3. 根据数据 Value 设置颜色 (例如: 蓝 -> 黄 -> 红)
            // 这里简单演示，value 越大越红
            colorTool.setHSL(0.6 - item.value * 0.6, 1.0, 0.5); 
            colors[i * 3] = colorTool.r;
            colors[i * 3 + 1] = colorTool.g;
            colors[i * 3 + 2] = colorTool.b;
        });

        // 将颜色属性分配给 InstancedMesh
        this.heatPointsMesh.geometry.setAttribute('instanceColor', new THREE.InstancedBufferAttribute(colors, 3));
        this.group.add(this.heatPointsMesh);
    }

    // ==========================================
    // 2. 动态飞线实现 (简化 Shader 版)
    // ==========================================
    addFlyLine(start, end, color = 0xffaa00) {
        const vStart = this._latLonToVector3(start.lat, start.lon, this.radius);
        const vEnd = this._latLonToVector3(end.lat, end.lon, this.radius);
        
        // 计算贝塞尔曲线控制点 (两点中点，并在更高半径上)
        const vCenter = vStart.clone().add(vEnd).multiplyScalar(0.5).normalize().multiplyScalar(this.radius * 1.5);
        
        const curve = new THREE.QuadraticBezierCurve3(vStart, vCenter, vEnd);
        const points = curve.getPoints(100); // 采样 100 个点
        
        const geometry = new THREE.BufferGeometry().setFromPoints(points);
        
        // 用于Shader的Progress属性，0.0-1.0代表点在曲线上的位置
        const percents = new Float32Array(points.length);
        for(let i=0; i<points.length; i++) {
            percents[i] = i / points.length;
        }
        geometry.setAttribute('aPercent', new THREE.BufferAttribute(percents, 1));

        // 核心：飞线 Shader
        const material = new THREE.ShaderMaterial({
            uniforms: {
                uTime: { value: 0 },
                uColor: { value: new THREE.Color(color) },
                uLength: { value: 0.2 } // 光斑占整条线的比例
            },
            vertexShader: `
                attribute float aPercent;
                varying float vPercent;
                void main() {
                    vPercent = aPercent;
                    gl_Position = projectionMatrix * modelViewMatrix * vec4(position, 1.0);
                }
            `,
            fragmentShader: `
                uniform vec3 uColor;
                uniform float uTime;
                uniform float uLength;
                varying float vPercent;
                void main() {
                    // 创建循环流动的光斑逻辑
                    float head = fract(uTime * 0.5); // 0.0 -> 1.0 循环
                    float opacity = 0.0;
                    
                    if(vPercent > head - uLength && vPercent < head) {
                        // 在光斑区域内，计算尾部渐变
                        opacity = (vPercent - (head - uLength)) / uLength;
                    }
                    
                    gl_FragColor = vec4(uColor, opacity);
                }
            `,
            transparent: true,
            blending: THREE.AdditiveBlending,
            depthTest: false
        });

        const line = new THREE.Line(geometry, material);
        this.group.add(line);
        this.flyLines.push(line);
    }

    // 需要在主循环中调用
    update(time) {
        this.flyLines.forEach(line => {
            line.material.uniforms.uTime.value = time;
        });
    }
}
```

## 🟠 模块四：交互管理与局部特写 (Interaction & Camera Fly-to)

本模块的目标是将之前所有的静态/动态图层通过**相机控制**串联起来。实现“指哪打哪”的特写效果，并让缩放过程伴随粒子的聚拢/散开特效。

### 💻 核心实现思路

1. **点击检测 (Raycaster)**：
    
    - 在 3D 空间中，鼠标点击的是 2D 屏幕坐标。我们需要通过 `THREE.Raycaster` 从相机发射射线，检测与地球（或数据点）的交点，获取经纬度。
        
2. **平滑特写 (Camera Fly-to)**：
    
    - 直接修改相机坐标会很生硬。我们使用 **GSAP** 曲线动画，同时改变相机位置（Position）和视点中心（OrbitControls.target）。
        
3. **缩放与粒子特效联动 (Uniform Linkage)**：
    
    - **关键点**：监听相机的 `distance`。当相机靠近地球表面时，动态修改模块一中 `ShaderMaterial` 的 `uZoom` 值，实现“越近越聚拢”的精细化视觉体验。
        

### 💻 核心代码框架 (交互管理器)

JavaScript

```
import * as THREE from 'three';
import gsap from 'gsap';

export default class InteractionManager {
    constructor(earthEngine, dataLayer) {
        this.engine = earthEngine; // 引用主引擎
        this.dataLayer = dataLayer;
        this.raycaster = new THREE.Raycaster();
        this.mouse = new THREE.Vector2();
        
        this.initListeners();
    }

    initListeners() {
        window.addEventListener('click', (e) => this.onMouseClick(e));
        
        // 实时监听缩放，联动粒子特效
        this.engine.controls.addEventListener('change', () => {
            this.handleZoomSync();
        });
    }

    // 1. 同步相机距离与粒子聚拢系数
    handleZoomSync() {
        const dist = this.engine.camera.position.length();
        // 映射逻辑：距离越小(近)，uZoom越大(聚拢)
        // 假设正常距离为400，特写距离为120
        const zoomFactor = THREE.MathUtils.mapLinear(dist, 120, 400, 2.5, 1.0);
        this.engine.particleMaterial.uniforms.uZoom.value = THREE.MathUtils.clamp(zoomFactor, 1.0, 2.5);
    }

    // 2. 点击拾取
    onMouseClick(event) {
        this.mouse.x = (event.clientX / window.innerWidth) * 2 - 1;
        this.mouse.y = -(event.clientY / window.innerHeight) * 2 + 1;

        this.raycaster.setFromCamera(this.mouse, this.engine.camera);
        
        // 检测与热力图点或透明代理球体的碰撞
        const intersects = this.raycaster.intersectObject(this.engine.earth);

        if (intersects.length > 0) {
            const point = intersects[0].point;
            this.focusOnPoint(point);
        }
    }

    // 3. 局部特写动画 (Fly-to)
    focusOnPoint(targetVector3) {
        // 计算相机目标位置：在交点方向上稍微向外延伸一段距离
        const targetPos = targetVector3.clone().normalize().multiplyScalar(150); 

        // 停止当前的控制器交互
        this.engine.controls.enabled = false;

        const tl = gsap.timeline({
            onComplete: () => {
                this.engine.controls.enabled = true;
            }
        });

        // 动画 A：相机位置移动
        tl.to(this.engine.camera.position, {
            x: targetPos.x,
            y: targetPos.y,
            z: targetPos.z,
            duration: 1.5,
            ease: "power2.inOut"
        }, 0);

        // 动画 B：视点中心同步移动到点击点
        tl.to(this.engine.controls.target, {
            x: targetVector3.x,
            y: targetVector3.y,
            z: targetVector3.z,
            duration: 1.5,
            ease: "power2.inOut",
            onUpdate: () => {
                this.engine.camera.lookAt(this.engine.controls.target);
            }
        }, 0);
    }
}
```

## 🟣 模块五：后期特效与系统集成 (Post-processing & Final Integration)

本模块的目标是提升视觉品质，实现“电影感”的辉光效果，并完成模块化的最终封装。

### 💻 核心实现思路

1. **辉光特效 (Bloom Effect)**：
    
    - 在 3D 渲染中，粒子和飞线如果只是单纯的颜色，会显得比较“平”。
        
    - 我们引入 **`EffectComposer`** 和 **`UnrealBloomPass`**。它会识别画面中亮度超过阈值的部分（如我们的粒子和光斑），并对其进行高斯模糊后再叠加上去，产生梦幻的发光感。
        
2. **环境氛围 (Starfield)**：
    
    - 创建一个巨大的反向球体或使用 `THREE.Points` 随机分布在极远处，模拟星空背景，增加空间深度。
        
3. **模块总装 (The Entry Point)**：
    
    - 创建一个 `World` 类，作为对外暴露的唯一接口。
        

### 💻 核心代码框架 (后期处理与集成)

JavaScript

```
import * as THREE from 'three';
import { EffectComposer } from 'three/examples/jsm/postprocessing/EffectComposer';
import { RenderPass } from 'three/examples/jsm/postprocessing/RenderPass';
import { UnrealBloomPass } from 'three/examples/jsm/postprocessing/UnrealBloomPass';

// 假设之前的模块已经分别导出为 EarthEngine, GeoLayer, DataLayer, InteractionManager

export default class ParticleEarthApp {
    constructor(containerId) {
        this.container = document.getElementById(containerId);
        
        // 1. 初始化基础引擎
        this.engine = new EarthEngine(this.container);
        
        // 2. 初始化图层
        this.geoLayer = new GeoLayer(this.engine.scene, 100);
        this.dataLayer = new DataLayer(this.engine.scene, 100);
        
        // 3. 初始化交互控制
        this.interaction = new InteractionManager(this.engine, this.dataLayer);

        // 4. 配置后期辉光
        this.initPostProcessing();
        
        // 5. 添加星空背景
        this.addStarfield();

        // 启动循环
        this.render();
    }

    initPostProcessing() {
        const renderScene = new RenderPass(this.engine.scene, this.engine.camera);

        // 参数：分辨率、强度、半径、阈值
        this.bloomPass = new UnrealBloomPass(
            new THREE.Vector2(window.innerWidth, window.innerHeight),
            1.5,  // 辉光强度
            0.4,  // 半径
            0.85  // 阈值：只有亮部发光
        );

        this.composer = new EffectComposer(this.engine.renderer);
        this.composer.addPass(renderScene);
        this.composer.addPass(this.bloomPass);
    }

    addStarfield() {
        const vertices = [];
        for (let i = 0; i < 5000; i++) {
            const x = THREE.MathUtils.randFloatSpread(2000);
            const y = THREE.MathUtils.randFloatSpread(2000);
            const z = THREE.MathUtils.randFloatSpread(2000);
            vertices.push(x, y, z);
        }
        const geometry = new THREE.BufferGeometry();
        geometry.setAttribute('position', new THREE.Float32BufferAttribute(vertices, 3));
        const material = new THREE.PointsMaterial({ color: 0x888888, size: 0.7 });
        const stars = new THREE.Points(geometry, material);
        this.engine.scene.add(stars);
    }

    render() {
        const time = performance.now() * 0.001;
        
        // 更新各模块内部逻辑
        this.engine.update(); // 处理 Controls
        this.dataLayer.update(time); // 处理飞线动画
        
        // 使用 Composer 代替原生的 renderer.render
        this.composer.render();
        
        requestAnimationFrame(() => this.render());
    }

    // 对外 API：方便后续扩展
    addDataPoint(lat, lon, value) {
        this.dataLayer.createHeatmapPoints([{lat, lon, value}]);
    }

    addConnection(start, end) {
        this.dataLayer.addFlyLine(start, end);
    }
}
```
