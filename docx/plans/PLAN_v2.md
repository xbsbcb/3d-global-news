## 状态机设计

| 状态 | 粒子行为 | 边界线行为 | 相机行为 |
|------|---------|-----------|---------|
| **Normal** | 跟随 earthGroup | 跟随 earthGroup | OrbitControls，缓慢自转 |
| **Focused** | 静止（暂时禁用动画） | 跟随 earthGroup | 飞向目标，camera.lookAt 持续更新朝向 |

---

## 动画时序

### 聚焦动画 (1.5s)
- 放大先慢后快：`ease: 'power2.in'`
- 相机沿法线方向移动到 distance
- 持续使用 `camera.lookAt(surfacePoint)` 更新朝向

### 回位动画 (0.8s)
- 缩小先快后慢：`ease: 'power2.out'`
- 相机回归 (0, 0, 400)
- 持续使用 `camera.lookAt(0, 0, 0)` 更新朝向

---

## 聚焦距离计算

```typescript
// 基于地理范围计算
const latSpan = bounds.maxLat - bounds.minLat
const lngSpan = bounds.maxLng - bounds.minLng
const span = Math.max(latSpan, lngSpan, 5)
const distance = Math.max(100, Math.min(250, 400 / span))
```

## 分层设计

| 层级 | 高度偏移 | 说明 |
|------|---------|------|
| 粒子层 | 0 | 跟随 earthGroup |
| 填充层 | 0.5 | 贴合球面，三角剖分 |
| 边界线层 | 2.0 | 位于填充上方，呈现分层感 |

## 交互设计

| 操作 | 功能 |
|------|------|
| 左键点击 | 聚焦到点击位置（聚焦状态忽略） |
| 右键点击 | 取消聚焦，回归默认 |
| 拖拽 | OrbitControls 旋转/缩放 |
| 普通状态 | 缓慢自动旋转 |
| 上下拖拽后 | 2秒无操作自动回正到中立位置 |
