## 使用说明

### 一键测试/构建

```bash
# 测试全部模块
./scripts/build.sh all

# 测试指定模块
./scripts/build.sh frontend.particle-earth  # 粒子地球
./scripts/build.sh backend.api             # 后端API
```

### Git 提交

每次 `git commit` 会自动：
1. 生成生产报告到 `docx/生产报告/`
2. 显示提交统计信息

### 模块完成流程

1. 完成模块开发
2. 运行 `./scripts/build.sh <module>` 进行测试构建
3. 提交代码 `git add . && git commit -m "完成XXX模块"`
4. 自动更新模块状态报告
